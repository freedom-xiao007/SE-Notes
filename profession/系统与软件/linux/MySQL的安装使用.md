## 一、MySQL及其图形操作界面的命令行安装
&ensp;&ensp;&ensp;&ensp;输入下面的命令进行安装即可，默认安装会开机启动，如果不需要可以按照下面的链接进行设置，MySQL安装会提示输入root用户密码：

```
sudo apt-get install mysql-server-5.7
sudo apt-get install mysql-workbench
```
