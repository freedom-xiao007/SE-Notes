## 启动使用
&ensp;&ensp;&ensp;&ensp;openvas采用docker方式进行运行，首先运行下面的命令启动其相应的docker后进行docker。

&ensp;&ensp;&ensp;&ensp;进入docker以后运行相应的命令启动相应的服务：

```
openvassd -a 0.0.0.0 -p 9392
openvasmd -a 0.0.0.0 -p 9393
```
