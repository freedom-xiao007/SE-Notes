# 排序
***
### 字典排序
```sh
sorted(dataOrigin.items(), key=lambda k: k[1], reverse=False)
```

### 列表字典排序
```sh
import operator
x = [{'name':'Homer', 'age':39}, {'name':'Bart', 'age':10}]
sorted_x = sorted(x, key=operator.itemgetter('age'),reverse=True) # True 是倒叙  默认是False
print sorted_x
```
