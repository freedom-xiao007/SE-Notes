# 博客搭建
***

```shell script
F:\Software\hugo_0.80.0_Windows-64bit\hugo.exe server --theme=dimension --buildDrafts

F:\Software\hugo_0.80.0_Windows-64bit\hugo --theme=dimension --baseUrl="http://free-love.gitee.io/xiao.github.io/"
```
